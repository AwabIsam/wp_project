About

StrongPal is a fitness focused application with helps users set workouts they want to do so that they can hold themselves accountable and make sure they build up unbreakable discipline.

The StrongPal application hopes to achieve this by also allowing users to view their friends workout plan and hold each-other accountable for their workouts.

How to use:

  • Add workout.
  • Set Due Date.
  • Set Priority.
  • Once complete toggle as complete.
  • If you want to search up friends click on the search bar and entry their username.
  • You can add friends, or just view other peoples workouts.
  • Add a friend so that you can easily access them in the friends tab.
  • Make sure your friends always complete their workouts.
  • Have a great pump!
